# Home Assistant Add-ons: The unofficial repository

These add-ons are a proof of concept to integrate with or allow access to your Home Assistant configuration based on example add-on provided by Home Assistant community.

## Installation

To use this repository with your own Hass.io installation please follow [the official instructions](https://www.home-assistant.io/hassio/installing_third_party_addons/) on the Home Assistant website with the following URL:

```txt
https://github.com/fcojperez/ha-addons
```

## Add-ons provided by this repository

- **[Vault](./vault/README.md)**

    Vault Server to manage secrets and protect sensitive data
